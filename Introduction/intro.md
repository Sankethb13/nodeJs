# Introduction to Nodejs
    Nodejs was built in year 2009, It was developed by RYAN DAHL. 
    Here we will learn what is nodejs,

## What is NodeJs?
    - Nodejs is a javascript runtime and it is built on chrome V8 engine.
    - Nodejs is a crossplatform library. It can be runned on any OS.
    - It is a open-source runtime environment, It is maintained by "Open Js foundation"(Community which maintains node js).
    - Nodejs runs on v8 javascript engine. Which helps to execute Javascript code outside the web browser.
    - Whereever there is javascript there is JS engine. 

## History of Nodejs:
    - 2009 (Ryan Dahl built Nodejs) started with spiderMonkey (Firefox) it is not developed directly in the google v8 engine.
     later after 2 days he moved to v* engine.
    - Joyent (Company who worked on nodejs with Ryan)
    - Initial name is webjs named by Ryan, Later he named it to nodejs as they do not want to limit to web applications.
    - Problem: Firstly when they are creating a server using apache and http server. These were blocking servers. He wanted to create and make them non blocking. Hence they started developing nodejs/webjs.
    - 2010: NPM was introduced. package manager for node. "Joyent" made this open source so that everyone can contribute to the nodejs registry. 
    - 2011: Initially nodejs was developed only on macos and linux. Then in 2011 "Joyent and Microsoft" together added support to nodejs on windows.
    - 2012: This project was handed over to "ISAAC"(who is the developer of NPM).
    - 2014: "Fedor" developed similar version of node which is called io.js.
    - 2015: Nodejs foundation was formed and handled both nodejs and io.js
    - 2019: JS foundation + Nodejs foundation merged and actively handling node which is openJS Foundation.
