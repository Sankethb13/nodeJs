# Basics of Nodejs

## What is a server:
    - Server is a computer or CPU which is location in cloud or other location. A computer which receives the request and process the response accordingly is called a server. 
    - Previously JS was used only on the browsers or clients. Now we can use JS in the server with help of node. Which is very powerful.
## What is V8 engine:
    - V8 is google open source, high performance Javascript and web-assembly engine.
    - Which is written in C++.
    - It is used in google chrome and in node js
    - It implement ECMAScript and webAssembly.
    - It runs on any platform. Windows, MacOS, Linux 
    - This can be embedded to any C++ application.
    - Nodejs is C++ application with V8 embedded to it. 
    - So on an overview, Server -> Nodejs -> V8Engine 
    - V8 is a Javascript engine which follows ECMAScript 
## What is ECMAScript: 
    - It is a standard for scripting languages. Including Javascript, Jscript and Webscript.
    - Js engines follows these standards.
    - It defines rules.
